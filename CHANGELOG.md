# Changelog

## 1.1.0

- `forever` mode — loops the bar indefinitely without requiring a `progress` value

## 1.0.0

Initial release.

- `progress` property (0–100)
- `cols` property — configurable character width
- `color` property — Pac-Man mouth color (default `#d35400`)
- `track-color` / `trackColor` — color of brackets, dashes, and dots
